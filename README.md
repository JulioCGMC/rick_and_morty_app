## How to Use

```
- Primeiramente, é necessário ter o flutter instalado e configurado na sua maquina

- Clone o repositório:
$ git clone https://github.com/JulioCGMC/rick_and_morty_app.git

- Entre na pasta:
$ cd rick_n_morty_app

- Instale as dependencias:
$ flutter pub get

- E rode: 
$ flutter run
```
